Simple dotfiles for Omarchy

## Quality Of Life
- Changed the Hyprland navigation to Vim key bindindings.

## Hotkeys 

### Navigating
|        **Hotkey**       |       **Function**       |
|:-----------------------:|:------------------------:|
| Super + H/J/K/L         | Move focus to window     |
| Super + Shift + H/J/K/L | Swap window with another |
| Super + Alt + H/J/K/L   | Resize window            |
| Super + Q               | Close window             |
| Super + Ctrl + J        | Toggle split window      |
| Super + Ctrl + T        | Toggle float window      |


### Launching Apps
| **Hotkey**       | **Function**      |
|------------------|-------------------|
| Super + T        | Open Terminal     |
| Super + CTRL + K | Open Hotkeys Menu |
